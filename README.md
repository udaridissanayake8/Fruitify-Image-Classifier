## Repository Structure

The project directory is organized as follows:

```
Fruitify-Image-Classifier/
├── data/
│   ├── images/
│   └── labels/
├── src/
│   ├── main.py
│   ├── utils.py
│   └── model/
├── tests/
│   └── test_main.py
├── README.md
└── requirements.txt
```

- **data/**: Contains all the datasets used for training and testing.
- **src/**: Source code of the application, including the main script, utility functions, and model-related files.
- **tests/**: Contains test scripts to verify the functionality of the application.
- **README.md**: Documentation for the project.
- **requirements.txt**: Lists the Python dependencies required to run the application.
